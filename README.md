## 📖Introduction

> Sasaran: Membuat templat sumber terbuka Admin Vuetify 3 terbaik.

Dibangun berdasarkan tema Vuetify yang elegan, kami telah menetapkan struktur proyek yang jelas dan efisien, mengintegrasikan kerangka kerja teknologi terkini. Proyek ini bertujuan untuk memenuhi berbagai persyaratan dan fitur teknis umum, sekaligus menggabungkan asisten AI untuk pengalaman yang lebih cerdas. Selain itu, kami memastikan bahwa semua halaman bersifat adaptif di berbagai perangkat, sehingga mencapai kompatibilitas lintas platform yang mulus.

## 👻Try it now!

```
git clone https://github.com/ifamyprogreta/lux-admin-vuetify3.git

cd lux-admin-vuetify3

npm install --global yarn

yarn install

yarn dev
```

## 👻Docker it!

1. Build the development environment image:

   ```
   docker-compose build dev
   ```
2. Start the development environment:

   ```
   docker-compose up dev
   ```
3. Build the production environment image:

   ```
   docker-compose build app
   ```
4. Start the production environment:

   ```
   docker-compose up app
   ```

Ini akan menyelesaikan masalah pembaruan waktu nyata. Jika Anda memiliki pertanyaan lain atau mengalami masalah lain, silakan beri tahu saya.

### 🔑Set ApiKey

Temukan file `.env.template` di direktori root, hapus akhiran `.template`, dan ganti `VITE_OPENAI_API_KEY`, `VITE_UNSPLASH_ACCESS_KEY`, dan `VITE_GITHUB_CLIENT_ID`, serta `VITE_TTS_KEY` dan `VITE_TTS_REGION` dengan kunci Anda sendiri.

> openai apikey： https://platform.openai.com/account/api-keys

> unsplash apikey： https://unsplash.com/oauth/applications

> github apikey： https://github.com/settings/tokens

> azure textToSpeech : https://speech.microsoft.com/
